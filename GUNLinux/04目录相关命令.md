### 通配符

| 通配符  | 含义                             |
| ------- | -------------------------------- |
| `*`     | 代表任意个字符                   |
| `?`     | 代表任意1个字符，至少1个         |
| `[]`    | 表示可以匹配字符组中任意一个     |
| `[abc]` | 匹配 a b c 中任意一个            |
| `[a-f]` | 匹配从 a 到 f 这个范围中任意一个 |

例子：

```shell
ls *1.txt #找以1结尾的txt文件
```
