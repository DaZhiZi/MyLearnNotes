## 学习规划

学了两年多的 Python 会用 Python 造一些小工具，但是一直没有深入学习过 Python ，所以对于高级特性以及面向对象学的非常薄弱，所以需要系统的学习一遍，买了个网课，然后大概按照进度来规划一下。

因为以下东西比较多，所以准备把不打算学的删去。
