## 替换空格

### 题目描述

请实现一个函数，将一个字符串中的每个空格替换成“%20”。例如，当字符串为We Are Happy.则经过替换之后的字符串为We%20Are%20Happy。

### 思路

正着找复杂度是$O(n^2)​$无法接受，倒着$O(n)​$扫一遍就好了。

### 代码

```cpp
#include <bits/stdc++.h>
using namespace std;
class Solution
{
  public:
    void replaceSpace(char *str, int length)
    {
        if (str == NULL || length <= 0)
            return;
        int n = length;
        int cnt = 0;
        for (int i = 0; i < n; i++)
            if (str[i] == ' ')
                cnt++;
        if (cnt == 0)
            return;
        int p2 = n + cnt * 2;
        str[p2] = '\0';
        p2--;
        int i = n - 1;
        while (i >= 0)
        {
            if (str[i] == ' ')
            {
                str[p2--] = '0';
                str[p2--] = '2';
                str[p2--] = '%';
            }
            else
                str[p2--] = str[i];
            i--;
        }
    }
};
int main()
{
    Solution ac;
    char s[20] = "we are happy !";
    ac.replaceSpace(s, strlen(s));
    printf("%s\n", s);
    return 0;
}
```
